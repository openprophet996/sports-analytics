凯利公式计算器
f* = (b·p − q) / b · 最优资金管理 · 多注并行
参数输入
公式说明
f* = (b × p − q) / b

• b = (1 − 市价) / 市价

• p = 你的胜率估计

• q = 1 − p

⚠️ 推荐半凯利 ×0.5 控制方差

f* < 0 → 负期望，不下注

f* > 0 → 正期望，可下注

EV 期望值说明
EV = 净赔率×胜率 − 1×败率

EV = 2.33×0.40 − 1×0.60
